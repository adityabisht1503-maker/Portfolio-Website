
# Portfolio Website

A modern and responsive personal portfolio website built with **React** and **Vite** to showcase my skills, projects, achievements, and professional journey. The website provides visitors with an overview of my technical expertise, featured projects, and contact information through a clean and interactive user interface.

## 🌐 Live Demo

**Website:** https://portfolio-website-delta-cyan-55.vercel.app/

---

## ✨ Features

* Modern and responsive UI
* Interactive project showcase
* Skills and technology section
* About Me section
* Contact section
* Smooth navigation
* Fast performance with Vite
* Mobile-friendly design
* Clean and minimal layout

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/your-username/Portfolio-Website.git
```

```bash
cd Portfolio-Website
```

### Install dependencies

```bash
npm install
```

---

## ▶️ Running the Application

Start the development server:

```bash
npm run dev
```

Build the project:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
